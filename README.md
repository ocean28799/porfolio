## Getting Started

### Prerequisites
- Node.js (v18 or newer recommended)
- pnpm (or npm/yarn)

### Installation
```bash
pnpm install
```

### Development
```bash
pnpm dev
```

### Build
```bash
pnpm build
```

### Start
```bash
pnpm start
```


## Customization
- Update content in the `src/` directory (about, experience, projects, etc.)
- Replace images and assets in the `public/` folder
